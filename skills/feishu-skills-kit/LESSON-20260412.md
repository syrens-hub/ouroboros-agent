## Lesson #7: 套件Skill不是已安装Skills

**发现问题日期**: 2026-04-12

**现象**: `feishu-skills-kit` 列了10个子Skill，但只装了2个

**根因**: 套件类Skill是"技能目录/手册"，不是"已激活技能"

**实际状态**:
- 已安装 ✅: feishu-bitable / feishu-sheets-skill
- 未安装 ⚠️: feishu-bridge / feishu-card / feishu-doc-* / feishu-leave-request / feishu-memory-recall / feishu-messaging

**行动**: 使用套件Skill前，先确认目标子Skill是否已安装
```bash
ls ~/.openclaw/workspace/skills/ | grep feishu
```

**预防**: `clawhub install` 时，哪个需要装哪个，不要默认装全套
