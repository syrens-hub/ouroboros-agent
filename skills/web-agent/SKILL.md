# Web Agent Skill - 网页浏览与内容获取

## 功能
- 网页内容抓取（支持普通网站）
- JavaScript 渲染页面（需要浏览器）
- 内容提取与分析
- 支持的平台：微博、知乎、微信公众号等

## 使用场景
- 抓取公开网页内容
- 分析网页结构
- 提取文章/视频信息

## 限制
- 抖音、快手等有强反爬保护的平台需要用户手动提供内容
- 需要登录的页面需要 cookies

## 使用方法
当需要浏览网页或获取内容时：
1. 尝试直接抓取公开页面
2. 如果失败，提示用户提供内容或授权

## 技术方案
- requests + beautifulsoup4：普通网页
- Playwright：JavaScript 渲染页面
- API 调用：移动端接口（如果可用）
