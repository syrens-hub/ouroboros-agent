---

## 错误记录

### [ERR-20260406-001]
- **错误**: 任务描述过于笼统
- **原因**: 没使用标准化模板
- **后果**: Agent 交付物不符合预期，返工
- **修复**: 使用任务模板补充完整验收标准
- **预防**: 任务下发前强制检查完整性

### [ERR-20260406-002]
- **错误**: 多 Agent 并发时未定义好接口
- **原因**: 直接分发任务，没做协调
- **后果**: 输出格式不统一，汇总困难
- **修复**: 先定义协作接口规范，再分发任务
- **预防**: monitor.sh checkpoint 强制检查接口

### [ERR-20260406-003]
- **错误**: 复杂任务一次性下发
- **原因**: 没做任务分解
- **后果**: Agent 超出处理能力，交付失败
- **修复**: 拆分为 2-3 个阶段，分段交付
- **预防**: L3 复杂任务必须经过 Research Agent 拆解

### [ERR-20260406-004]
- **错误**: 任务完成后没验证直接交付
- **原因**: 缺少验收清单
- **后果**: 交付质量差，用户不满
- **修复**: 对照清单逐项检查，不合格退回
- **预防**: monitor.sh complete 前必须验收

### [ERR-20260406-005]
- **错误**: Agent 选了不擅长的方向
- **原因**: 凭直觉选型，没用 task-parser.sh
- **后果**: 交付质量差，需换 Agent 重做
- **修复**: 任务下发前用 parser.sh 验证
- **预防**: 复杂任务强制走解析流程
